# Forge Spaces

Sistema web para cadastro, gerenciamento e reserva de espaços para eventos, desenvolvido em ASP.NET Web Forms.

## Sobre o projeto

O ForgeSpaces é uma aplicação web que permite aos usuários cadastrar espaços para locação,
visualizar opções disponíveis e realizar reservas.
O projeto foi desenvolvido com foco em aprendizado prático de ASP.NET Web Forms, C# e SQL Server.

## Funcionalidades

- Cadastro e login de usuários.
- Cadastro de espaços com upload de imagem.
- Listagem de espaços.
- Gerenciamento de espaços do usuário.
- Realização de reservas.
- Gerenciamento de reservas do usuário.

## Tecnologias utilizadas

- ASP.NET Web Forms
- C#
- SQL Server
- Bootstrap
- HTML, CSS e JavaScript

## Como executar o projeto

1. Clone o repositório.
2. Abra o projeto no Visual Studio.
3. Caso o banco esteja com problema, criar nova base de dados e rodar o script "Criacao_Banco.sql"
4. Configure a string de conexão no `Web.config`.
5. Execute o projeto com IIS Express.
